同頁多個swiper使用及pagination、navigation的控制

    HTML

  1. 分別將第一個swiper的class改成：swiper1、swiper2。
  2. 為了易於辨識，可以先將第二組的navigation用inline style的方式改成其他顏色

    3. CSS

  3. CSS中 .swier1及 .swiper2中，除了寬高、顏色之外，需要設定一個position: relative。
    因為pagination、navigation在js中會自動找到重新選取的swiper區塊，但css的設定不會，
    若在Chrome中檢查pagination及navigation，就會發現他們預設了一個position: absolute的屬性，
    所以並非是js的設定有問題，而是兩個區塊的button重疊在一起了，導致點選到的都是最後一個區塊的上下頁按鈕。

    4. JS

  4. 修改初始化const swiper的部分，分別命名為
    const swiper1 = new Swiper('.swiper1',...
    const swiper2 = new Swiper('.swiper2',...
